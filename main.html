사용법

  1. 단어장 텍스트 파일을 드래그&드롭 하거나 파일 선택 버튼을 눌러 직접 불러옵니다.
    단어장 파일은 아래와 같은 형식입니다. 

    apple
    사과
banana
    바나나
cinnamon
    계피

...

[단어]
    [뜻]

  2. 원하는 설정을 한 후에, 원하는 테스트를 진행하면 됩니다.
  • 단어 읽어주기의 경우 현재 영단어의 원형만 가능하며, Merriam-Webster online dictionary의 API 를 사용합니다.
